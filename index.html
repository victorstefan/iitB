A web page (or webpage) is a web document that is suitable for the World Wide Web and the web browser.

A web browser displays a web page on a monitor or mobile device. The web page is what displays, but the term also refers to a computer file, usually written in HTML or comparable markup language. Web browsers coordinate the various web resource elements for the written web page, such as style sheets, scripts and images, to present the web page.

Typical web pages provide hypertext that include a navigation bar or a sidebar menu to other web pages via hyperlinks, often referred to as links.
